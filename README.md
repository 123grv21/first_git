# Git Practice Repository
This repository is for practicing Git and version control concepts. It includes hands-on exercises using essential Git commands and workflows.

## Objectives
- Understand version control basics
- Practice Git commands safely
- Learn collaboration through branches and merges

## Topics Covered
- git init – Initialize a new repository
- git clone – Copy a remote repository
- git add, git commit – Track and record changes
- git status, git log – View repository state and history
- git branch – Create and manage branches
- git checkout, git switch – Move between branches
- git merge – Combine work from different branches
- Merge conflict resolution
- git pull, git fetch, git push – Sync with remote
- .gitignore usage
- Reverting and resetting commits
- Working with remotes (origin, upstream)
